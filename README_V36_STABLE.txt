6. Actual calculator page loads all calculator options from content/site-content.json.
7. Password login kept.
8. Admin Password block added.
9. Full top navigation restored.
10. GitHub Pages direct publish kept.
11. No Vercel API.

Install:
1. Unzip this file.
2. Replace your GitHub repo files with this version.
3. Delete api/publish.js if it exists.
4. Commit changes.
5. Open https://stnlysee.github.io/siongdesign/admin/index.html
6. Press Ctrl + F5.
7. Login with current password. Default fallback is siongadmin.
