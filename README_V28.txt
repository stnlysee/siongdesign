Siong Design V28 Admin + Calculator + FAQ Update

This is based on the easy GitHub Pages version, not full backend.

Added:
1. Page preview selector inside admin
   - Home
   - Services
   - Why Us
   - Works
   - Calculator
   - FAQ
   - Contact

2. Page Content editor
   - Edit page headings/descriptions
   - Add/remove cards for Services, Why Us, Works etc.

3. FAQ Editor
   - Add FAQ
   - Remove FAQ
   - Edit question and answer
   - Drag/reorder FAQ

4. Calculator category editor
   - HDB category
   - Condo category
   - Add/remove category
   - Add/remove pricing options inside each category

5. Better default FAQ content
   - Why use melamine board instead of cheap PVC board?
   - What plywood thickness do we use? 6mm plywood backing, 9mm solid plywood internal shelving

6. Scroll fix
   - Left panel scroll
   - Middle preview scroll
   - Right editor panel scroll

7. GitHub Pages publish fix
   - Removed cache-control/no-store GitHub API issue
   - Image file upload is still disabled for GitHub Pages stability
   - To change actual images, upload/replace them manually in assets folder

Install:
1. Unzip this file.
2. Replace your current website files/folders with these files.
3. Commit changes to GitHub.
4. Delete api/publish.js if it exists.
5. Open https://stnlysee.github.io/siongdesign/admin/index.html
6. Press Ctrl + F5.
7. Login password: siongadmin.
